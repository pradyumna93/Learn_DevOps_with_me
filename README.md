# Learn DevOps With Me

A modern, vibrant blog for DevOps beginners and enthusiasts!  
Check out the [blog here](https://pradyumna93.github.io/Learn_devops_with_me/) (after you enable GitHub Pages).

## Features

- Colorful, engaging design
- Beginner-friendly DevOps blog posts
- Easy to extend with new posts

## Getting Started

1. Clone the repo.
2. Add your own posts in the `posts/` folder.
3. Enable GitHub Pages from the `main` branch in repo settings.
